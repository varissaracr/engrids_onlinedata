ระบบบริการข้อมูลสารสนเทศ - EnGRIDs DSC


ระบบบริการฐานข้อมูลสารสนเทศ - EnGRIDs DSC โดยสตูดิโอวิจัยและพัฒนานวัตกรรมเชิงพื้นที่ด้านสิ่งแวดล้อม หรือเรียกว่า "EnGRIDs"
พัฒนาขึ้นเพื่อให้นักศึกษาของภาควิชาภูมิศาสตร์ มหาวิทยาลัยเชียงใหม่ สามารถเข้าถึงข้อมูลจากแหล่งที่มาที่มีความน่าเชื่อถือ ทั้งรูปแบบของข้อมูลเชิงพื้นที่ (Shapefile),
ข้อมูล API จากหน่วยงานภาครัฐที่เปิดให้บริการ รวมถึงข้อมูลสารสนเทศอื่นๆ เพื่อให้เกิดความสะดวก และรวดเร็ว ในการใช้งาน
